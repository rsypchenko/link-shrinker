# Link Shrinker

This is a URL shortener Node.js service.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the server:
   ```sh
   node index.js
   ```
3. Open your browser and go to [http://localhost:3000](http://localhost:3000)

## Requirements
- Node.js
- npm
